uiselect

文档

组件化开发

使用此控件需要把items 对象中的文本定义成text,id定义成id 不能为其他

对象默认为items 可在 :items = 替换为自己的对象名称

选中时，uiselectid 就是选中的id,uiselecteindex 就是选中的索引,uiselecttext 就是选中的文本

灵活性：可调整hover 颜色 ，class（宽度，高度建议保持不变，圆角等）style。 建议点击后的蓝色边框不要随时调整，麻烦

局限性:对象中items 必须设置id名称为id，文本名称为text，isshowhover：false， 也可以根据需要去组件中调整

uiselectid: {{uiselectid}}

uiselecttext: {{uiselecttext}}
